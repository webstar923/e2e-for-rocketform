# Installation

### Clone the repository to your local machine:
    git clone https://github.com/webstar923/e2e-for-rocketform.git

### Clone the repository to your local machine:
    cd e2e-for-rocketform

### Install dependencies
    npm install

### Run the program
    npm start

# Technology Stack
